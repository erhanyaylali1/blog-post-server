# Blog-Post

DESCRIPTION
--------------------------------
- A blog with rich text editor

Features
--------------------------------
- Rich Text Editor
- Create Post
- Edit Post
- Add Categories to Posts
- Add Tags to Posts
- Upload Image
- Like to Post
- Comment to Post
- View Count

Technologies
--------------------------------
- Next JS
- Node JS
- Mongo DB with Mongoose
- Swagger io
- Google Cloud (to do)
- Bootstrap, Material UI, Ant Design
